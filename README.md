# java-calculator
Simple calculator using Java (Console based).
🧮 Java Calculator

A simple console-based calculator built using Java.

✨ Features

- Addition
- Subtraction
- Multiplication
- Division
- Handles divide-by-zero errors

🛠️ Technologies Used

- Java
- OOP concepts
- User input handling (Scanner)

▶️ How to Run

1. Compile the program:
   javac Calculator.java

2. Run:
   java Calculator
